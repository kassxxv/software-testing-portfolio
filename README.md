### About This Portfolio

This project serves as a practical demonstration of the skills I developed 
during an Automation Testing in Python course. Feeling that the coursework 
alone wasn't enough, I decided to build a full test framework from scratch, 
manually, using AI only as a reference tool.

**What's inside:**
- **Databases:** SQL, PostgreSQL, SQLite, SQLAlchemy, PonyORM
- **API/UI Testing:** Playwright, Requests, Pytest, Pydantic, Faker, Mocks
- **Other:** Core Python libraries, Logging, CSS/XPath selectors, Docker, 
  Jenkins, CI/CD, Allure
